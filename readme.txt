== Description ==

This is an unofficial prayer time widget based on JAKIM e-solat API, default location is set to Kuala Lumpur.

A default CSS is provided. However you're free to modify the styling based on your site requirement.

TODO: Add the ability to use current user geolocation.

== Installation ==

1. Upload the plugin to your WordPress site.
2. Activate the plugin.
3. Set the widget options like location, widget title, calculation method
4. Enjoy
